# Credit Risk Modeling
Developed a Credit Risk Modeling system using Logistic Regression with feature engineering (e.g., credit utilization rate), deployed via FastAPI and Streamlit to assess loan default risk in real-time.

## 📌 Overview
This project predicts whether a customer is a high-risk or low-risk borrower using machine learning. It helps financial institutions make better lending decisions.

## 🚀 Features
- Real-time credit risk prediction
- Feature engineering (credit utilization rate)
- FastAPI backend for predictions
- Streamlit frontend for easy interaction

## 🧠 Model
- Algorithm: Logistic Regression
- Includes scaling and preprocessing

## 🔍 Feature Engineering
- Credit Utilization Rate = CreditAmount / MaxCreditAmount
- Age grouping
- Encoding categorical variables

## 🛠 Tech Stack
- Python
- scikit-learn
- FastAPI
- Streamlit
- Pandas, NumPy

## 📊 Output
- Classifies customer as:
  - High Risk
  - Low Risk
- Provides prediction probability
